# K. Запросы композиции перестановок

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 512 мегабайт

Вам дан массив $a_1, \dots, a_n$, состоящий из перестановок длины $m$.

Мы можем определить операцию $\cdot$ для двух перестановок $x$ и $y$ длины $m$ как такую перестановку $z = x \cdot y$,
что $z_i = y_{x_i}$ для всех $1 \le i \le m$. Заметьте, что порядок перемножения важен.

Вам даны $q$ запросов, каждый запрос задается двумя числами $1 \le l \le r \le n$. Рассмотрим перестановку $b$
длины $m$, такую что $b = ((\dots((a_l \cdot a_{l+1}) \cdot a_{l+2}) \cdot \dots) \cdot a_r)$. Тогда ответом на запрос
будет являться сумма $\sum_{i=1}^{m} i \cdot b_i$. Реализуйте программу, быстро отвечающую на эти запросы.

### Входные данные

Первая строка содержит единственное целое число $t$ равное количеству тестовых случаев ($1 \le t \le 1000$). Далее
следует описание $t$ тестовых случаев, каждое в следующем формате:

Первая строка каждого описания содержит два целых числа $n$, $m$ ($1 \le n, m \le 10^5$
и $1 \le n \cdot m \le 2 \cdot 10^5$). Следующие $n$ строк содержат по $m$ различных целых
чисел $a_{i1}, a_{i2}, \dots, a_{iM}$, разделенных пробелами ($1 \le a_{ij} \le m$). Следующая строка содержит
единственное целое число $q$ ($1 \le q \le 2 \cdot 10^5$). Следующие $q$ строк содержат по два целых числа $l$, $r$,
разделенных пробелами ($1 \le l \le r \le n$).

Гарантируется, что сумма $n \cdot m$ и сумма $q$ по всем тестовым случаям не превосходит $2 \cdot 10^5$.

### Выходные данные

Выведите ответы на запросы в том порядке, в котором они заданы во входных данных.
