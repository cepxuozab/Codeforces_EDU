# C. Прибавь на прямоугольнике

**ограничение по времени на тест:** 3 секунды  
**ограничение по памяти на тест:** 256 мегабайт

Дан двумерный массив $a$. Поступают запросы прибавления константы на подпрямоугольниках этого массива. В конце нужно
вывести получившийся массив.

### Входные данные

Первая строка входных данных содержит два целых числа $n$ и $m$ ($1 \le n, m \le 1000$) — размерности массива $a$.

Каждая из последующих $n$ строк содержит по $m$ целых чисел $a_{i,j}$ ($-10^9 \le a_{i,j} \le 10^9$) — элементы
массива $a$.

Следующая строка входных данных содержит одно целое число $q$ ($1 \le q \le 10^6$) — количество запросов прибавления на
прямоугольнике.

Каждая из последующих $q$ строк содержит по пять целых чисел $lx_i$, $ly_i$, $rx_i$, $ry_i$
и $d_i$ ($1 \le lx_i \le rx_i \le n$, $1 \le ly_i \le ry_i \le m$, $-10^9 \le d_i \le 10^9$) — границы подпрямоугольника
массива, на котором надо прибавить константу, а также сама константа.

### Выходные данные

Выведите $n$ строк по $m$ чисел в каждой — элементы массива после выполнения всех операций.

### Примеры

**Ввод:**

```text
1 1
10
3
1 1 1 1 100
1 1 1 1 1
1 1 1 1 -3
```

**Вывод:**

```text
108
```

**Ввод:**

```text
2 2
0 0
0 0
3
1 1 2 2 5
1 1 1 2 4
2 1 2 1 -8
```

**Вывод:**

```text
9 9 
-3 5 
```

**Ввод:**

```text
3 3
1 2 3
4 5 6
7 8 -9
3
3 3 3 3 18
1 1 3 3 5
1 2 2 3 -2
```

**Вывод:**

```text
6 5 6 
9 8 9 
12 13 14 
```
