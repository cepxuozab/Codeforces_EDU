# A. Найдите сумму на отрезке

**Ограничение по времени на тест:** 3 секунды  
**Ограничение по памяти на тест:** 256 мегабайт

Дан массив $a$. Поступают запросы поиска суммы на отрезке. Вам необходимо на них отвечать.

## Входные данные

Первая строка входных данных содержит одно целое число $n$ ($1 \le n \le 10^6$) — количество элементов массива.

Вторая строка входных данных содержит $n$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$) — элементы массива $a$.

Третья строка входных данных содержит одно целое число $q$ ($1 \le q \le 10^6$) — количество запросов поиска суммы на
отрезке.

Каждая из последующих $q$ строк содержит по два целых числа $l_i$ и $r_i$ ($1 \le l_i \le r_i \le n$) — границы отрезка
массива, на котором нужно найти сумму.

## Выходные данные

Для каждого запроса в отдельной строке выведите сумму элементов на соответствующем отрезке массива (включая границы).

### Примеры

**Ввод:**

```text
2
10 30
3
1 1
2 2
1 2
```

**Вывод:**

```text
10
30
40
```

**Ввод:**

```text
5
-1 -2 3 0 4
5
1 5
1 2
4 4
3 4
2 5
```

**Вывод:**

```text
4
-3
0
3
5
```

**Ввод:**

```text
7
1 1 1 1 1 1 1
9
1 7
2 6
3 4
2 5
5 7
6 6
3 3
2 7
1 1
```

**Вывод:**

```text
7
5
2
4
3
1
1
6
1
```
