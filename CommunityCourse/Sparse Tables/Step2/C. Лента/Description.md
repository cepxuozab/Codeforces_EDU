## C. Лента

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт

У Александры есть лента, а на ней — $n$ чисел. Обозначим их за $a_i$ слева направо.

Теперь Александра хочет разделить ленту на несколько частей (возможно, на $1$). Для каждой части должны выполняться
следующие условия:

- Часть должна содержать не менее $l$ чисел.
- Разница между наибольшим и наименьшим числом в части должна быть не более $s$.

Пожалуйста, помогите Александре найти минимальное количество частей, на которые можно разделить ленту, соблюдая
указанные условия.

### Входные данные

В первой строке записано три разделённых пробелами целых
числа $n$, $s$, $l$ ($1 \le n \le 10^5$, $0 \le s \le 10^9$, $1 \le l \le 10^5$).

Во второй строке следуют $n$ разделённых пробелами целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

### Выходные данные

Выведите минимальное количество частей, на которое можно разделить ленту.

Если способа разделить ленту не существует, выведите $-1$.

### Пример

**Входные данные:**

```text
7 2 2
1 3 1 2 4 1 2
```

**Выходные данные:**

```text
3
```

**Входные данные:**

```text
7 2 2
1 100 1 100 1 100 1
```

**Выходные данные:**

```text
-1
```  

**Примечание**  
В первом примере можно разделить ленту на 3 части: [1, 3, 1], [2, 4], [1, 2].

Во втором примере нельзя поместить 1 и 100 на одну часть, так что решения не существует.
