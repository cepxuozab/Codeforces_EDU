## F. НОД на отрезке

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 256 мегабайт

Вам дан массив из $n$ чисел и $q$ запросов. Каждый запрос задается двумя числами $i$ и $j$; ответом на каждый запрос
является наибольший общий делитель (НОД) элементов массива на отрезке $[i, j]$ (включительно).

Наибольший общий делитель (НОД) двух целых чисел $a$ и $b$ — это наибольшее положительное целое число, которое делит
нацело как $a$, так и $b$.

### Входные данные

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит одно целое
число $t$ ($1 \le t \le 10^4$) — количество наборов входных данных. Далее следует описание наборов входных данных.

Первая строка каждого набора входных данных содержит одно целое число $n$ ($1 \le n \le 5 \times 10^5$) — количество
элементов в массиве.

Вторая строка каждого набора входных данных содержит $n$ целых чисел $a_0, a_1, \dots, a_{n-1}$ ($0 \le a_i \le 10^9$) —
элементы массива.

Третья строка каждого набора входных данных содержит одно целое число $q$ ($1 \le q \le 5 \times 10^5$) — количество
запросов.

Каждая из следующих $q$ строк содержит два целых числа $i$ и $j$ ($0 \le i \le j \le n-1$) — границы отрезка в $0$
-индексации.

Гарантируется, что сумма $n$ и сумма $q$ по всем наборам входных данных не превосходят $5 \times 10^5$.

### Выходные данные

Для каждого набора входных данных выведите $q$ строк: в $k$-й из них выведите ответ на $k$-й запрос.

### Пример

**Входные данные**

```text
2
5
2 4 6 8 10
1
1 3
4
3 5 9 15
2
0 2
2 3
```

**Выходные данные:**

```text
2
1
3
```

### Примечание

В первом наборе входных данных массив имеет длину $5$: $[2, 4, 6, 8, 10]$. Есть $1$ запрос, запрашивающий НОД подмассива
с индекса $1$ по индекс $3$ (включительно). Подмассив — это $[4, 6, 8]$, поэтому ответ $\gcd(4,6,8) = 2$.

Во втором наборе входных данных массив имеет длину $4$: $[3, 5, 9, 15]$. Есть $2$ запроса:

- Первый запрос запрашивает НОД подмассива $[0, 2]$: то есть $\gcd(3,5,9) = 1$.
- Второй запрос запрашивает НОД подмассива $[2, 3]$: то есть $\gcd(9,15) = 3$.
