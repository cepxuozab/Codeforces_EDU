# A. Произведение на отрезке по модулю

**Ограничение по времени на тест:** 7 секунд  
**Ограничение по памяти на тест:** 256 мегабайт

Вам дан массив $a_0, a_1, \dots, a_{n-1}$ из $n$ целых чисел и целое число $p$ (не обязательно простое). Необходимо
ответить на $q$ запросов.

$i$-й запрос определяется двумя целыми числами $l_i$ и $r_i$. Ответом на запрос является произведение элементов

$$
a_{l_i} \cdot a_{l_i+1} \cdot \dots \cdot a_{r_i} \pmod{p}.
$$

Вы должны отвечать на запросы в **онлайн-режиме**, то есть не можете узнать следующий запрос, пока не ответите на все
предыдущие.

## Входные данные

Каждый тест состоит из нескольких наборов входных данных.

Первая строка содержит одно целое число $t$ ($1 \le t \le 100$) — количество наборов входных данных. Далее следует
описание наборов.

Каждый набор входных данных состоит из трёх строк:

1. Три целых числа $n$, $p$, $q$ ($2 \le n \le 10^6$, $2 \le p \le 10^9$, $1 \le q \le 2 \cdot 10^7$).
2. Вторая строка содержит $n$ целых чисел $a_0, a_1, \dots, a_{n-1}$ ($0 \le a_i \le 10^9$).
3. Третья строка содержит $\lfloor \frac{q}{64} \rfloor + 2$ целых чисел $b_0, b_1, \dots$ ($0 \le b_i \le n-1$).

Запросы нумеруются от $0$ до $q-1$.

### Формирование запросов

Для запроса $i$, где $i$ делится на $64$:

$$
l_i \equiv b_{\lfloor i/64 \rfloor} + x \pmod{n}, \quad
r_i \equiv b_{\lfloor i/64 \rfloor + 1} + x \pmod{n}.
$$

В противном случае:

$$
l_i \equiv l_{i-1} + x \pmod{n}, \quad
r_i \equiv r_{i-1} + x \pmod{n}.
$$

Если $l_i > r_i$, нужно поменять их местами.

Для **нулевого** запроса $x = 0$.  
Для всех остальных запросов $x$ равен **ответу на предыдущий запрос** $+ 1$, взятому по модулю $p$.

### Гарантии

Сумма $n$ по всем наборам входных данных не превышает $10^6$, а сумма $q$ не превышает $2 \cdot 10^7$.

## Выходные данные

Для каждого набора входных данных выведите одно целое число — значение $x$ **после последнего запроса**, взятое по
модулю $p$, после применения всех предыдущих запросов.

## Пример

**Входные данные:**

```text
2
5 11 2
1 2 3 4 5
0 1
4 17 1
6 7 8 9
3 2
```

**Выходные данные:**

```text
10
5
```

## Примечание к примеру

### Набор 1

- **Запрос 0** ($i=0$ делится на 64):  
  $x = 0$,  
  $l_0 \equiv b_0 + x \pmod{5} = 0$,  
  $r_0 \equiv b_1 + x \pmod{5} = 1$.  
  Произведение $= a[0] \cdot a[1] = 1 \cdot 2 = 2$,  
  $x = (2 + 1) \bmod 11 = 3$.

- **Запрос 1**:  
  $l_1 \equiv l_0 + x \pmod{5} = 0 + 3 = 3$,  
  $r_1 \equiv r_0 + x \pmod{5} = 1 + 3 = 4$.  
  Произведение $= a[3] \cdot a[4] = 4 \cdot 5 = 20$,  
  $x = (20 + 1) \bmod 11 = 21 \bmod 11 = 10$.

**Ответ:** $10$.

### Набор 2

- **Запрос 0**:  
  $x = 0$,  
  $l_0 \equiv b_0 + x \pmod{4} = 3$,  
  $r_0 \equiv b_1 + x \pmod{4} = 2$.  
  Меняем местами: $l_0 = 2, r_0 = 3$.  
  Произведение $= a[2] \cdot a[3] = 8 \cdot 9 = 72$,  
  $x = (72 + 1) \bmod 17 = 73 \bmod 17 = 5$.

**Ответ:** $5$.
