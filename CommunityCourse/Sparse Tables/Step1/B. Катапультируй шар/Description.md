## B. Катапультируй шар

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 1024 мегабайта

В некоторой стране существует $n$ городов, расположенных вдоль прямой. В каждом городе есть ровно один холм. Высота
холма в городе $i$ обозначается как $h_i$.

Вам нужно ответить на $m$ запросов. Для каждого запроса нужно выяснить, возможно ли отправить магический шар из
города $a$ в город $b$.

Доставка из города $a$ в город $b$ возможна, если каждый холм строго между городами $a$ и $b$ (с индексами от $a+1$
до $b-1$ включительно) имеет высоту, меньшую или равную высоте холма в городе $a$.

### Входные данные

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит целое число $t$ ($1 \le t \le 1000$) —
количество наборов входных данных.

Первая строка каждого набора входных данных содержит два целых числа $n$
и $m$ ($2 \le n \le 10^5$, $1 \le m \le 10^5$) — количество городов и количество запросов.

Вторая строка каждого набора входных данных содержит $n$ целых чисел $h_1, h_2, \dots, h_n$ ($1 \le h_i \le 10^9$) —
высоты холмов.

Каждая из следующих $m$ строк содержит два целых числа $a$ и $b$ ($1 \le a \le b \le n$) — описание очередного запроса
доставки шара из города $a$ в город $b$.

Гарантируется, что сумма $n$ и сумма $m$ по всем наборам входных данных не превосходят $10^5$.

### Выходные данные

Для каждого набора входных данных выведите одно целое число — количество доставок, которые могут быть выполнены успешно.

### Пример

**Входные данные:**

```text
2
7 3
2 3 5 4 2 1 6
3 5
2 5
4 6
2 1
1 2
1 2
```

**Выходные данные:**

```text
2
1
```

**Примечание**  
В первом наборе входных данных могут быть выполнены доставки 1 и 3.

Во втором наборе входных данных доставка может быть выполнена, поскольку между исходным и целевым городами нет холмов.
