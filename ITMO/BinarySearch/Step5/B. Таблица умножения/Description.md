# B. Таблица умножения

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт

Петя составил таблицу умножения размера $n \times n$. Ячейка в $i$-й строке и $j$-м столбце содержит
значение $i \cdot j$. Петю заинтересовал вопрос: какое число в таблице является $k$-м по возрастанию? Помогите Пете
ответить на этот вопрос.

## Входные данные

Ввод содержит два целых числа $n$ и $k$ ($1 \le n \le 10^5$, $1 \le k \le n^2$).

## Выходные данные

Выведите одно число — $k$-е число по возрастанию в таблице.

### Примеры

**Ввод**

```
3 5
```

**Вывод**

```
3
```

**Ввод**

```
5 16
```

**Вывод**

```
10
```
