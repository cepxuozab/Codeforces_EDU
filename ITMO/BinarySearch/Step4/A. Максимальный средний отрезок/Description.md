## A. Максимальный средний отрезок

Дан массив из n чисел и число d . Требуется найти отрезок массива длины не менее d , на котором среднее арифметическое
элементов максимально.

### Формат ввода

Первая строка содержит два целых числа n и d ($1 \le d \le n \le 10^5$). Следующая строка содержит n
чисел $a_i$ ($0 \le a_i \le 100$) — элементы массива.

### Формат вывода

Выведите два числа — номера первого и последнего элемента требуемого отрезка (элементы нумеруются с 1).

### Примеры

**Ввод**

```
6 2
3 1 8 5 7 2
```

**Вывод**

```
3 5
```
