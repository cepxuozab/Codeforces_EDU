# B. Минимальный средний путь

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 512 мегабайт

Дорожная сеть состоит из $n$ перекрестков и $m$ односторонних дорог, при этом каждая дорога ведет из перекрестка с
меньшим номером в перекресток с большим номером. Каждой дороге соответствует некоторое число. Требуется найти путь из
перекрестка $1$ в перекресток $n$, на котором среднее арифметическое чисел, соответствующих ребрам, минимально.

## Входные данные

Первая строка содержит целые числа $n$ и $m$ ($2 \le n \le 10^5$, $1 \le m \le 10^5$). Следующие $m$ строк содержат
тройки чисел $a_i$, $b_i$, $c_i$ ($1 \le a_i < b_i \le n$, $0 \le c_i \le 100$), которые означают, что есть дорога,
ведущая из перекрестка $a_i$ в перекресток $b_i$, которой соответствует число $c_i$. Для каждой пары перекрестков
существует не более одной дороги, которая их соединяет. Гарантируется, что существует путь из перекрестка $1$ в
перекресток $n$.

## Выходные данные

В первой строке выведите число ребер в выбранном пути $k$. В следующей строке выведите $k+1$ число — номера
перекрестков, которые посещает выбранный путь.

### Примеры

**Ввод**

```
4 3
1 2 1
2 3 0
2 4 1
```

**Вывод**

```
2
1 2 4 
```

**Ввод**

```
3 3
1 2 1
2 3 2
1 3 1
```

**Вывод**

```
1
1 3 
```
