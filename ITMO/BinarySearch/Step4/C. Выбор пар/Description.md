# C. Выбор пар

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Заданы $n$ пар положительных целых чисел $(a_1, b_1), (a_2, b_2), \dots, (a_n, b_n)$. Выберите $k$ ($1 \le k \le n$) из
них $i_1, i_2, \dots, i_k$, чтобы отношение
$$ \frac{a_{i_1} + a_{i_2} + \dots + a_{i_k}}{b_{i_1} + b_{i_2} + \dots + b_{i_k}} $$
было максимальным возможным.

## Входные данные

В первой строке записаны целые числа $n$ и $k$ ($1 \le k \le n \le 10^5$). Следующие $n$ строк содержат пары целых
чисел $a_i, b_i$ ($1 \le a_i, b_i \le 10^5$).

## Выходные данные

Выведите искомое максимальное отношение. Ответ будет считаться верным, если относительная или абсолютная погрешность не
превышает $10^{-6}$.

### Примеры

**Ввод**

```
3 2
10 3
9 5
7 4
```

**Вывод**

```
2.4285714286
```

**Ввод**

```
8 3
4 2
4 2
2 2
1 5
5 3
3 5
2 2
5 3
```

**Вывод**

```
1.8571428571
```
