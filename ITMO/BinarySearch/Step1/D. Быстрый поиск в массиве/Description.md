# D. Быстрый поиск в массиве

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 512 мегабайт

Дан массив $a$ из $n$ целых чисел $a_1, a_2, \dots, a_n$.

Нужно уметь отвечать на запросы вида «Сколько чисел имеют значения от $l$ до $r$»?

## Входные данные

В первой строке находится целое число $n$ ($1 \le n \le 10^5$) — длина массива.

Во второй строке находятся $n$ целых чисел $a_1, a_2, \dots, a_n$ ($-10^9 \le a_i \le 10^9$).

В третьей строке находится целое число $k$ ($1 \le k \le 10^5$) — число запросов.

В следующих $k$ строках находятся пары чисел $l$, $r$ ($-10^9 \le l \le r \le 10^9$).

## Выходные данные

Выведите $k$ чисел — ответы на запросы.

### Примеры

**Ввод**

```
5
10 1 10 3 4
4
1 10
2 9
3 4
2 2
```

**Вывод**

```
5 2 2 0 
```
