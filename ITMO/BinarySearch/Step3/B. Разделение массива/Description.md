## B. Разделение массива

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Дан массив из n положительных целых чисел. Нужно разбить его на k отрезков так, чтобы максимальная сумма на отрезке была
минимально возможной.

### Входные данные

Первая строка содержит целые числа n и k ($1 \leq k \leq n \leq 10^5$). Вторая строка содержит элементы
массива $a_i$ ($1 \leq a_i \leq 10^9$).

### Выходные данные

Выведите одно число — минимально возможную максимальную сумму на отрезке.

## Примеры

**Ввод**

```
10 4
1 3 2 4 10 8 4 2 5 3
```

**Вывод**

```
12
```
