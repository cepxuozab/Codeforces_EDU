## C. Коровы в стойла

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

На прямой расположены стойла, в которые необходимо расставить коров так, чтобы минимальное расcтояние между коровами
было как можно больше.

### Входные данные

В первой строке вводятся числа n ($ 2 \leq n \leq 10^4$) — количество стойл и k ($ 2 \leq n \leq n$) — количество коров.
Во второй строке задаются n целых чисел в порядке возрастания — координаты стойл (координаты находятся в диапазоне от 0
до $10^9$, включительно).

### Выходные данные

Выведите одно число — наибольшее возможное минимальное расстояние между двумя коровами.

## Примеры

**Ввод**

```
6 3
2 5 7 11 15 20
```

**Вывод**

```
9
```
