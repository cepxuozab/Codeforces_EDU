# B. Веревочки

Есть n  веревочек, нужно нарезать из них k  кусков одинаковой длины. Найдите максимальную длину кусков,
которые можно получить.

## Входные данные
В первой строке заданы два числа — n и k  (1 &le; n, k &le; 10000).
Далее в каждой из последующих n строк записано по одному числу — длине очередной веревочки a<sub>i</sub>  (1 &le; a<sub>i</sub> &le; 10<sup>7</sup>).

## Выходные данные
Выведите одно вещественное число — максимальную длину кусков, которые можно получить.
Ответ будет считаться верным, если относительная или абсолютная погрешность не превышает 10<sup>-6</sup>.

### Пример 1

**Ввод**  
```
4 11
802
743
457
539
```

**Вывод**  
```
200.5
```
