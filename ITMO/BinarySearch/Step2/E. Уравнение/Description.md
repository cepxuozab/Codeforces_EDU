# E. Уравнение

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 256 мегабайт  

Найдите такое число $x$, что  

$$x^2 + \sqrt{x} = c$$

## Входные данные

В единственной строке содержится вещественное число $c$ ($1.0 \le c \le 10^{10}$).

## Выходные данные

Выведите одно число — искомый $x$.  
Ответ будет признан верным, если относительная или абсолютная погрешность не более $10^{-6}$.

## Пример
**входные данные**
```
2.0
```

**выходные данные**
```
1.0
```

**входные данные**
```
15.6
```

**выходные данные**
```
3.698232168829691
```
