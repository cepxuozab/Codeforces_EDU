# B. Умножение и сумма

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Есть массив из $n$ элементов, изначально заполненный единицами. Вам нужно написать структуру данных, которая
обрабатывает два вида запросов:

- умножить все элементы от $l$ до $r-1$ на число $v$;
- найти сумму на отрезке от $l$ до $r-1$.

Обе операции выполняются по модулю $10^9 + 7$.

## Входные данные

Первая строка содержит два числа $n$ и $m$ ($1 \leq n, m \leq 100000$) — размер массива и число операций. Далее следует
описание операций. Описание каждой операции имеет следующий вид:

- `1 l r v` — умножить все элементы от $l$ до $r-1$ на число $v$ ($0 \leq l < r \leq n$, $1 \leq v < 10^9 + 7$);
- `2 l r` — найти сумму на отрезке от $l$ до $r-1$ ($0 \leq l < r \leq n$).

## Выходные данные

Для каждой операции второго типа выведите соответствующее значение.

## Примеры

### Пример 1

**Входные данные:**

```
5 6
1 0 3 3
2 1 2
1 1 4 4
2 1 3
2 1 4
2 3 5
```

**Выходные данные:**

```
3
24
28
5
```

### Пример 2

**Входные данные:**

```
2 3
1 0 1 1000000
1 0 2 1000000
2 0 2
```

**Выходные данные:**

```
993000
```
