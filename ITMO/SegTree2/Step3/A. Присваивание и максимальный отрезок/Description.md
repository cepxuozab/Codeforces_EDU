## A. Присваивание и максимальный отрезок

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

---

Есть массив из $n$ элементов, изначально заполненный нулями. Вам нужно написать структуру данных, которая обрабатывает
два вида запросов:

* присвоить всем элементам на отрезке от $l$ до $r-1$ значение $v$,
* найти подотрезок с максимальной суммой.

---

### Входные данные

Первая строка содержит два числа $n$ и $m$ ($1 \leq n, m \leq 100000$) — размер массива и число операций. Далее следует
описание операций. Описание каждой операции имеет следующий вид:

* `l r v` — присвоить всем элементам на отрезке от $l$ до $r-1$
  значение $v$ ($0 \leq l < r \leq n$, $-10^9 \leq v \leq 10^9$).

### Выходные данные

Выведите $m$ строк: максимальную сумму чисел на подотрезке после каждой операции. Обратите внимание, что этот подотрезок
может быть пустым (при этом сумма на нём будет равна $0$).

## Примеры

**Ввод**

```
5 3
0 5 3
1 3 -1
3 4 -5
```

**Вывод**

```
15
7
3
```
