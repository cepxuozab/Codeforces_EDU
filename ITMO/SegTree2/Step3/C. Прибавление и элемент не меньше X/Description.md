## C. Прибавление и элемент не меньше X

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

---

Есть массив из $n$ чисел, изначально заполненный нулями. Вам нужно написать структуру данных, которая обрабатывает два
вида запросов:

* прибавить $v$ ко всем числам на отрезке от $l$ до $r-1$,
* найти минимальный $j$, для которого $j \geq l$ и $a[j] \geq x$.

---

### Входные данные

Первая строка содержит два числа $n$ и $m$ ($1 \leq n, m \leq 100000$) — размер массива и число операций. Далее следует
описание операций. Описание каждой операции имеет следующий вид:

* `1 l r v` — прибавить $v$ ко всем числам на отрезке от $l$ до $r-1$ ($0 \leq l < r \leq n$, $0 \leq v \leq 10^4$).
* `2 x l` — найти минимальный $j$, для которого $j \geq l$ и $a[j] \geq x$ ($0 \leq x \leq 10^9$, $0 \leq l < n$). Если
  такого нет, вывести $-1$.

Индексы начинаются с $0$.

### Выходные данные

Для каждой операции второго типа выведите ответ на запрос.

## Примеры

**Ввод**

```
5 7
1 2 5 3
1 1 3 4
2 3 0
2 5 0
1 4 5 5
2 5 3
2 8 1
```

**Вывод**

```
1
2
4
4
```
