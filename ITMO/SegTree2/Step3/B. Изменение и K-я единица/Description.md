## B. Изменение и K-я единица

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

---

Есть массив из $n$ булеанов, изначально заполненный нулями. Вам нужно написать структуру данных, которая обрабатывает
два вида запросов:

* изменить значение всех элементов на отрезке от $l$ до $r-1$ на противоположное,
* найти индекс $k$-й единицы.

---

### Входные данные

Первая строка содержит два числа $n$ и $m$ ($1 \leq n, m \leq 100000$) — размер массива и число операций. Далее следует
описание операций. Описание каждой операции имеет следующий вид:

* `1 l r` — изменить значение всех элементов на отрезке от $l$ до $r-1$ на противоположное ($0 \leq l < r \leq n$).
* `2 k` — найти $k$-ю единицу (единицы нумеруются с $0$, гарантируется, что в массиве достаточное количество единиц).

### Выходные данные

Для каждой операции второго типа выведите индекс соответствующей единицы (все индексы в этой задаче от $0$).

## Примеры

**Ввод**

```
5 7
1 1 3
2 1
1 0 2
2 0
2 1
1 0 5
2 2
```

**Вывод**

```
2
0
2
4
```
