# B. Применение MAX к отрезку

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Есть массив из $n$ элементов, изначально заполненный нулями. Вам нужно написать структуру данных, которая обрабатывает
два вида запросов:

1. Для всех $i$ от $l$ до $r-1$ сделать операцию $a_i = \max(a_i, v)$,
2. Узнать текущее значение в ячейке $i$.

## Входные данные

Первая строка содержит два числа $n$ и $m$ ($1 \leq n, m \leq 100000$) — размер массива и число операций. Далее следует
описание операций. Описание каждой операции имеет следующий вид:

- `1 l r v` — для всех $i$ от $l$ до $r-1$ сделать
  операцию $a_i = \max(a_i, v)$ ($0 \leq l < r \leq n$, $0 \leq v \leq 10^9$).
- `2 i` — найти значение в ячейке с индексом $i$ ($0 \leq i < n$).

## Выходные данные

Для каждой операции второго типа выведите соответствующее значение.

## Примеры

### Пример 1

**Входные данные:**

```
5 5
1 0 3 3
2 1
1 2 4 4
2 3
2 4
```

**Выходные данные:**

```
3
4
0
```
