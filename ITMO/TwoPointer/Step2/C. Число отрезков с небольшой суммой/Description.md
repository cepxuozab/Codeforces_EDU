# C. Число отрезков с небольшой суммой

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Дан массив из $n$ чисел $a_i$. Скажем, что отрезок этого массива $a[l..r]$ $(1 \le l \le r \le n)$ хороший, если сумма
чисел на этом отрезке не больше $s$. Ваша задача — найти число различных хороших отрезков.

## Входные данные

Первая строка содержит целые числа $n$ и $s$ $(1 \le n \le 10^5, 1 \le s \le 10^{18})$.  
Вторая строка содержит числа $a_i$ $(1 \le a_i \le 10^9)$.

## Выходные данные

Выведите одно число — число хороших отрезков.

---

### Пример 1

**Ввод:**

```text
7 20
2 6 4 3 6 8 9
```

**Вывод:**

```text
19
```
