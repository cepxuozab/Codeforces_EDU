# G. Взаимно простой отрезок

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Дан массив из $n$ чисел $a_i$. Скажем, что отрезок этого массива $a[l..r]$ $(1 \le l \le r \le n)$ хороший, если на этом
отрезке НОД всех чисел равен $1$. Ваша задача — найти самый короткий хороший отрезок.

## Входные данные

Первая строка содержит целое число $n$ $(1 \le n \le 10^5)$.  
Вторая строка содержит числа $a_i$ $(1 \le a_i \le 10^{18})$.

## Выходные данные

Выведите одно число — минимальную длину хорошего отрезка. Если хорошего отрезка не существует, выведите $-1$.

---

### Пример 1

**Ввод:**

```text
5
4 6 9 3 6
```

**Вывод:**

```text
3
```
