# B. Число меньших

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт

Даны два массива, отсортированные по неубыванию. Для каждого элемента второго массива найдите число элементов первого
массива, строго меньших его.

## Входные данные

В первой строке записаны числа $n$ и $m$ — размеры массивов $(1 \le n, m \le 10^5)$.  
Во второй строке находятся $n$ чисел $a_i$ — элементы первого массива,  
в третьей строке находятся $m$ чисел $b_i$ — элементы второго массива $(-10^9 \le a_i, b_i \le 10^9)$.

## Выходные данные

Выведите $m$ чисел — число элементов первого массива, меньших, чем каждый из элементов второго массива.

---

### Пример 1

**Ввод:**

```text
6 7
1 6 9 13 18 18
2 3 8 13 15 21 25
```

**Вывод:**

```text
1 1 2 3 4 6 6 
```
