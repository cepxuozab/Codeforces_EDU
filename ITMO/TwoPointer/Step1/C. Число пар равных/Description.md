# C. Число пар равных

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт

Даны два массива $a$ и $b$, отсортированные по неубыванию. Найдите число пар $(i, j)$, для которых $a_i = b_j$.

## Входные данные

В первой строке записаны числа $n$ и $m$ — размеры массивов $(1 \le n, m \le 10^5)$.  
Во второй строке находятся $n$ чисел $a_i$ — элементы первого массива,  
в третьей строке находятся $m$ чисел $b_i$ — элементы второго массива $(-10^9 \le a_i, b_i \le 10^9)$.

## Выходные данные

Выведите одно число — ответ на задачу.

---

### Пример 1

**Ввод:**

```text
8 7
1 1 3 3 3 5 8 8
1 3 3 4 5 5 5
```

**Вывод:**

```text
11 
```
