# E. Рюкзак на отрезке

ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт

Дан массив из $n$ предметов, для каждого известен его вес $w_i$ и стоимость $c_i$.  
Нужно выбрать **отрезок** этого массива, суммарный вес на котором не больше $s$, а суммарная стоимость — максимальна.

**Входные данные**

В первой строке вводятся целые числа $n$ и $s$  
($1 \le n \le 10^5$, $1 \le s \le 10^9$).

Во второй строке вводятся $n$ целых чисел $w_i$ ($1 \le w_i \le 10^9$).

В третьей строке вводятся $n$ целых чисел $c_i$ ($1 \le c_i \le 10^9$).

**Выходные данные**

Выведите одно число — наибольшую стоимость предметов, которые можно положить в рюкзак.

## Пример

**Ввод**

```
6 20
9 7 6 5 8 4
7 1 3 6 8 3
```

Вывод

```
17
```
