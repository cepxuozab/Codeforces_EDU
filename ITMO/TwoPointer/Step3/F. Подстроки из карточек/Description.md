# F. Подстроки из карточек

* ограничение по времени на тест: 2 секунды
* ограничение по памяти на тест: 512 мегабайт

Дана строка $s$ и $m$ карточек с буквами. Ваша задача — посчитать число подстрок строки $s$, которые можно составить из
этих карточек.

Например, если $s = \text{«aaab»}$, и есть три карточки с буквами «a», «a» и «b», то можно составить три подстроки «a»,
подстроку «b», две подстроки «aa», подстроки «ab» и «aab». А подстроки «aaa» и «aaab» — нельзя, так как есть всего две
карточки с буквой «a».

**Входные данные**

Первая строка входных данных содержит два целых числа $n$ и $m$ ($1 \le n, m \le 10^5$) — длину строки и число карточек.

Вторая строка входных данных содержит строку $s$ длины $n$.

Третья строка входных данных содержит строку длины $m$, задающую буквы, записанные на карточках.

Обе строки состоят только из букв латинского алфавита в нижнем регистре.

**Выходные данные**

Выведите одно целое число — количество подстрок в $s$, которые можно составить из данных карточек.

## Пример

**Ввод**

```
4 3
aaab
aba
```

Вывод

```
8
```

**Ввод**

```
7 3
abacaba
abc
```

Вывод

```
15
```
