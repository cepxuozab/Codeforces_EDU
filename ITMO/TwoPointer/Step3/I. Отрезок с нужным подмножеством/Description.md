# I. Отрезок с нужным подмножеством

ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 1024 мегабайта

Дан массив из $n$ чисел $a_i$. Скажем, что отрезок этого массива $a[l \dots r]$ **хороший**, если на этом отрезке можно
выбрать некоторый набор чисел, сумма которых равна $s$.  
Ваша задача — найти самый короткий хороший отрезок.

**Входные данные**

Первая строка содержит целые числа $n$ и $s$ ($1 \le n \le 10^5$, $1 \le s \le 1000$).  
Вторая строка содержит числа $a_i$ ($1 \le a_i \le s$).

**Выходные данные**

Выведите одно число — минимальную длину хорошего отрезка. Если хорошего отрезка не существует, выведите $-1$.

## Пример

**Ввод**

```
10 100
14 33 22 21 11 5 13 28 61 2
```

Вывод

```
5
```
