# A. Неориентированный граф

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

Вам задано два числа $n$ и $m$ и список из $m$ пар целых чисел (каждое число в паре от $1$ до $n$). Проверьте, что
заданный список пар является корректным набором ребер некоторого неориентированного графа (в графе не должно быть петель
и кратных ребер).

## Входные данные

В первой строке находится целое число $t$ ($1 \le t \le 10^5$) — количество наборов входных данных в тесте. Далее
следуют $t$ наборов входных данных. Перед каждым набором входных в тесте записана пустая строка.

В первой строке каждого набора тестовых данных записаны два целых числа $n$
и $m$ ($1 \le n \le 10^5$, $0 \le m \le 10^5$) — ограничения на числа в парах и количество пар соответственно.

В каждой из следующих $m$ строк записана пара целых чисел $u$ и $v$ ($1 \le u, v \le n$).

Суммы $n$ и $m$ по всем наборам тестовых данных не превосходят $10^5$.

## Выходные данные

Для каждого набора тестовых данных в отдельной строке выведите:

- «YES» — если заданный список пар является корректным набором ребер некоторого неориентированного графа (полученный
  граф не имеет петель и кратных ребер);
- «NO» — в противном случае.

### Пример 1

#### Ввод

```
3


3 3
1 2
1 3
3 2


2 2
1 2
2 2


2 2
1 2
2 1
```

#### Вывод

```
YES
NO
NO
```
