# B. Степени вершин

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

Вам задан неориентированный граф списком своих рёбер. Найдите степени всех вершин графа.

## Входные данные

В первой строке находится целое число $t$ ($1 \le t \le 10^5$) — количество наборов входных данных в тесте. Далее
следуют $t$ наборов входных данных. Перед каждым набором входных в тесте записана пустая строка.

В первой строке каждого набора содержатся два целых числа $n$ и $m$ ($1 \le n \le 10^5$, $0 \le m \le 10^5$), где $n$ —
количество вершин в графе, $m$ — количество рёбер в графе.

В следующих $m$ строках записаны рёбра, по одному ребру в строке. Каждое ребро характеризуется двумя целыми числами $u$
и $v$ ($1 \le u, v \le n$), где $u$ — начало ребра, а $v$ — конец ребра.

В графе отсутствуют петли и кратные рёбра.

Суммы $n$ и $m$ по всем наборам тестовых данных в одном тесте не превосходят $10^5$.

## Выходные данные

Для каждого набора тестовых данных в отдельной строке выведите $n$ целых чисел, где $i$-е число является степенью $i$-й
вершины графа.

### Пример 1

#### Ввод

```
4

5 6
1 2
2 3
3 1
4 3
5 4
5 2

3 2
1 2
2 3

2 1
1 2

4 0
```

#### Вывод

```
2 3 3 2 2 
1 2 1 
1 1 
0 0 0 0 
```
