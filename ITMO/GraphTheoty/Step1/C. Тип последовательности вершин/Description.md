# C. Тип последовательности вершин

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

Вам заданы неориентированный граф и последовательность вершин. Граф задан списком его рёбер. Определите, является ли
данная последовательность вершин:

- простым циклом,
- циклом,
- простым путём,
- путём

или ничем из перечисленных.

## Входные данные

В первой строке находится целое число $t$ ($1 \le t \le 10^5$) — количество наборов входных данных в тесте. Далее
следуют $t$ наборов входных данных. Перед каждым набором входных в тесте записана пустая строка.

В первой строке каждого набора содержатся три целых числа $n$, $m$
и $k$ ($1 \le n \le 10^5$, $0 \le m \le 10^5$, $2 \le k \le 10^5$), где $n$ — количество вершин в графе, $m$ —
количество рёбер в графе, $k$ — длина последовательности вершин.

В следующей строке записана последовательность из $k$ чисел $a_1, \dots, a_k$ ($1 \le a_i \le n$) — последовательность
вершин.

В следующих $m$ строках записаны рёбра, по одному ребру в строке. Каждое ребро характеризуется двумя целыми числами $u$
и $v$ ($1 \le u, v \le n$), где $u$ — начало ребра, а $v$ — конец ребра.

В графе отсутствуют петли и кратные рёбра.

Суммы $n$, $m$ и $k$ по всем наборам входных данных в одном тесте не превосходят $10^5$.

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите тип последовательности вершин:

- Если последовательность вершин является простым циклом, то выведите «simple cycle»,
- Если последовательность вершин является циклом, то выведите «cycle»,
- Если последовательность вершин является простым путём, то выведите «simple path»,
- Если последовательность вершин является путём, то выведите «path»,
- Иначе выведите «none».

Если подходят несколько вариантов, то нужно выводить тот, который находится в списке раньше.
