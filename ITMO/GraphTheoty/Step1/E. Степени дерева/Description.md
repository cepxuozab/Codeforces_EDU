# E. Степени дерева

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

Вам задан массив $d$ состоящий из $n$ элементов. Проверьте, существует ли дерево у которого массив степеней совпадает с
массивом $d$.

Массив степеней — такой массив $d$, где для каждой вершины $u$ значение $d_u$ равно степени вершины $u$.

## Входные данные

В первой строке находится целое число $t$ ($1 \le t \le 10^5$) — количество наборов входных данных в тесте. Далее
следуют $t$ наборов входных данных. Перед каждым набором входных в тесте записана пустая строка.

В первой строке каждого набора находится одно целое число $n$ ($1 \le n \le 10^5$) — количество элементов в массиве $d$.

В следующей строке записаны $n$ чисел $d_1, d_2, \dots, d_n$ ($0 \le d_i \le 10^9$) — массив $d$.

Сумма $n$ по всем наборам входных данных в одном тесте не превосходят $10^5$.

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите:

- «YES» — если существует дерево, у которого массив степеней совпадает с массивом $d$;
- «NO» — в противном случае.

### Пример 1

#### Ввод

```
3


7
2 1 3 1 1 1 3


2
2 2


1
0
```

#### Вывод

```
YES
NO
YES
```
