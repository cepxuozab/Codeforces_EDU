# F. Граф по массиву степеней

* ограничение по времени на тест 3 секунды
* ограничение по памяти на тест 256 мегабайт

Вам дан массив степеней некоторого неориентированного графа. Постройте по заданному массиву любой неориентированный
граф.

В графе не могут быть петли или кратные рёбра.

Вершины графа нумеруются с $1$.

## Входные данные

В первой строке находится целое число $t$ ($1 \le t \le 1000$) — количество наборов входных данных в тесте. Далее
следуют $t$ наборов входных данных. Перед каждым набором входных в тесте записана пустая строка.

В первой строке каждого набора содержатся одно целое число $n$ ($1 \le n \le 1000$) — размер массива степеней.

В следующей строке записан массив из $n$ чисел $a_1, a_2, \dots, a_n$ ($0 \le a_i \le n-1$) — степени вершин.

Сумма $n$ по всем наборам входных данных в одном тесте не превосходят $1000$. Гарантируется, что по заданному массиву
степеней можно построить граф.

## Выходные данные

Для каждого набора входных данных выведите в отдельной строке количество рёбер в графе и сам граф в виде множества
рёбер.

Каждое ребро должно быть выведено в отдельной строке, как пара из двух чисел, записанных через пробел — номера вершин на
концах ребра. В построенном графе массив степеней должен поэлементно совпадать с заданным массивом степеней.

Если существует несколько решений, выведите любое.

### Пример 1

#### Ввод

```
6

3
2 2 2

4
3 2 3 2

3
1 2 1

3
0 1 1

4
2 2 2 0

1
0
```

#### Вывод

```
3
3 2
3 1
2 1
5
3 1
3 4
3 2
1 4
1 2
2
2 3
2 1
1
3 2
3
3 2
3 1
2 1
0
```
