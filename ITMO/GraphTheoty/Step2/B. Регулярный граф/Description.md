# B. Регулярный граф

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам заданы два целых числа $n$ и $k$. Постройте $k$-регулярный неориентированный граф из $n$ вершин или укажите, что его не существует.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 1000)$ — количество наборов тестовых данных в тесте. Далее следуют $t$ наборов тестовых данных.

Каждый набор тестовых данных состоит из одной строки, в которой записаны два целых числа $n$ и $k$ $(1 \le n, k \le 200)$.

**Сумма $n$ по всем наборам тестовых данных не превышает $1000$.**

## Выходные данные

Для каждого набора тестовых данных выведите:

- **«YES»** — если существует $k$-регулярный неориентированный граф из $n$ вершин.  
  В следующей строке выведите количество рёбер $m$ и в следующих $m$ строках сам граф в виде множества рёбер.  
  Каждое ребро должно быть выведено в отдельной строке, как пара из двух чисел, записанных через пробел — номеров вершин на концах ребра.  
  Если существует несколько решений, выведите любое.

- **«NO»** — в противном случае.

**Вершины графа нумеруются с единицы.**  

## Пример
**входные данные**
```
3
3 2
5 3
5 4
```

**выходные данные**
```
YES
3
1 2
2 3
3 1
NO
YES
10
1 2
1 3
2 3
2 4
3 4
3 5
4 5
4 1
5 1
5 2
```
