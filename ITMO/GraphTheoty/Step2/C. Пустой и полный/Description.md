# C. Пустой и полный

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам задан неориентированный граф списком его рёбер. Проверьте, что заданный граф является объединением не более одного пустого и не более одного полного графов.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 10^5)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных. **Перед каждым набором входных данных в тесте записана пустая строка.**

В первой строке каждого набора содержатся два целых числа $n$ и $m$ $(1 \le n \le 10^5,\ 0 \le m \le 10^5)$, где $n$ — количество вершин в графе, $m$ — количество рёбер в графе.

В следующих $m$ строках записаны рёбра, по одному ребру в строке. Каждое ребро характеризуется двумя целыми числами $u$ и $v$ $(1 \le u, v \le n)$, где $u$ — начало ребра, а $v$ — конец ребра.

В графе отсутствуют петли и кратные рёбра.

**Суммы $n$ и $m$ по всем наборам входных данных в одном тесте не превосходят $10^5$.**

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите:

- **«YES»** — если заданный граф является объединением не более одного пустого и не более одного полного графов;
- **«NO»** — в противном случае.   


## Пример
**входные данные**
```
2

4 3
2 3
2 4
3 4

4 3
1 2
2 3
3 4
```

**выходные данные**
```
YES
NO
```
