# A. Количество вершин

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт  

Вам задано целое неотрицательное число $m$. Выведите минимальное количество вершин неориентированного графа, который может иметь ровно $m$ ребер.  

## Входные данные

В первой строке находится целое число t ($1 \le t \le 10^5$) — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных.

Каждый набор входных данных состоит из одной строки, в которой записано целое число $m$ ($0 \le m \le 10^9$).

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите минимальное количество вершин неориентированного графа, который может иметь ровно $m$ ребер.

## Пример
**входные данные**
```
6
1
10
54
1241
914890921
1000000000
```

**выходные данные**
```
2
5
11
51
42777
44722
```
