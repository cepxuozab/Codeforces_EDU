# D. Вторые соседи

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам задан ориентированный граф списком его дуг. Для каждой вершины $v$ выведите все вершины, которые смежны хотя бы с одной вершиной, которая смежна с $v$, в возрастающем порядке.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 10)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных. **Перед каждым набором входных данных в тесте записана пустая строка.**

В первой строке каждого набора содержатся два целых числа $n$ и $m$ $(1 \le n \le 50,\ 0 \le m \le 200)$, где $n$ — количество вершин в графе, $m$ — количество дуг в графе.

В следующих $m$ строках записаны дуги, по одной дуге в строке. Каждая дуга характеризуется двумя целыми числами $u$ и $v$ $(1 \le u, v \le n)$, где $u$ — начало дуги, а $v$ — конец дуги (дуга направлена из $u$ в $v$).

В графе отсутствуют кратные дуги.

## Выходные данные

Для каждого набора входных данных выведите $n$ строк. В $i$-й строке выведите номера вершин, которые смежны хотя бы с одной вершиной, которая смежна с $i$-й, в возрастающем порядке.


## Пример
**входные данные**
```
1

7 8
1 2
2 3
2 6
2 7
5 2
4 2
7 4
6 5
```

**выходные данные**
```
3 6 7 
4 5 

3 6 7 
3 6 7 
2 
2 
```
