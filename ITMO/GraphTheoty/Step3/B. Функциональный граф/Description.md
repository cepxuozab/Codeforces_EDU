# B. Функциональный граф

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам задан ориентированный граф списком его дуг. За один ход можно либо добавить, либо удалить любую дугу из графа.

Граф называется **функциональным**, если степень исхода каждой вершины равна $1$.

Найдите минимальное количество ходов, которое требуется, чтобы получить из заданного графа функциональный.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 10^5)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных. **Перед каждым набором входных данных в тесте записана пустая строка.**

В первой строке каждого набора содержатся два целых числа $n$ и $m$ $(1 \le n \le 10^5,\ 0 \le m \le 10^5)$, где $n$ — количество вершин в графе, $m$ — количество дуг в графе.

В следующих $m$ строках записаны дуги, по одной дуге в строке. Каждая дуга характеризуется двумя целыми числами $u$ и $v$ $(1 \le u, v \le n)$, где $u$ — начало дуги, а $v$ — конец дуги (дуга направлена из $u$ в $v$).

В графе отсутствуют кратные дуги.

**Суммы $n$ и $m$ по всем наборам входных данных в одном тесте не превосходят $10^5$.**

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите одно целое число — минимальное количество ходов, которое требуется, чтобы получить из заданного графа функциональный.

## Пример
**входные данные**
```
2

4 4
1 2
2 3
1 4
4 3

4 2
1 2
3 4
```

**выходные данные**
```
2
2
```
