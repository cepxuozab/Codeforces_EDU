# A. Истоки и стоки

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам задан ориентированный граф списком его дуг.

**Истоком** ориентированного графа называется вершина $u$, для которой не существует вершины $v$, такой что $v$ смежна с $u$ (нет входящих в $u$ дуг).

**Стоком** ориентированного графа называется вершина $u$, для которой не существует вершины $v$, такой что $u$ смежна с $v$ (нет исходящих из $u$ дуг).

Посчитайте количество истоков и стоков заданного графа.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 10^5)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных. **Перед каждым набором входных данных в тесте записана пустая строка.**

В первой строке каждого набора содержатся два целых числа $n$ и $m$ $(1 \le n \le 10^5,\ 0 \le m \le 10^5)$, где $n$ — количество вершин в графе, $m$ — количество дуг в графе.

В следующих $m$ строках записаны дуги, по одной дуге в строке. Каждая дуга характеризуется двумя целыми числами $u$ и $v$ $(1 \le u, v \le n)$, где $u$ — начало дуги, а $v$ — конец дуги (дуга направлена из $u$ в $v$).

В графе отсутствуют кратные дуги.

**Суммы $n$ и $m$ по всем наборам входных данных в одном тесте не превосходят $10^5$.**

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите два целых числа, разделённых пробелом — количество истоков и стоков графа.

## Пример
**входные данные**
```
3

2 1
1 2

2 3
1 1
1 2
2 2

5 0
```

**выходные данные**
```
1 1
0 0
5 5
```
