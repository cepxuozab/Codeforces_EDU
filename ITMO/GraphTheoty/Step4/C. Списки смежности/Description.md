# C. Списки смежности

**Ограничение по времени на тест:** 3 секунды  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам задана матрица смежности некоторого неориентированного графа. Постройте по заданной матрице списки смежности.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 500)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных.

В первой строке каждого набора содержится одно целое число $n$ $(1 \le n \le 500)$ — размер матрицы.

В следующих $n$ строках записаны по $n$ чисел $a_{ij}$, разделённых через пробел — бинарная матрица. Каждое число является либо единицей, либо нулём.

**Сумма $n$ по всем наборам входных данных в одном тесте не превосходит $500$.**

## Выходные данные

Для каждого набора входных данных выведите $n$ строк. В $i$-й строке выведите все смежные с $i$-й вершины в порядке возрастания номеров. Вершины нумеруются с единицы.



## Пример
**входные данные**
```
2
2
0 1
1 0
3
0 0 1
0 0 1
1 1 0
```

**выходные данные**
```
2 
1 
3 
3 
1 2 
```
