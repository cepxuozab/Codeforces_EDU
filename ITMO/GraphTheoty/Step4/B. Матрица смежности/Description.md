# B. Матрица смежности?

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 256 мегабайт  

Вам заданы число $n$ и бинарная (состоящая из нулей и единиц) матрица размера $n \times n$. Проверьте, является ли заданная матрица матрицей смежности некоторого неориентированного графа и если это так, то выведите степени вершин этого графа.

## Входные данные

В первой строке находится целое число $t$ $(1 \le t \le 500)$ — количество наборов входных данных в тесте. Далее следуют $t$ наборов входных данных.

В первой строке каждого набора содержится одно целое число $n$ $(1 \le n \le 500)$ — размер матрицы.

В следующих $n$ строках записаны по $n$ чисел $a_{ij}$, разделённых через пробел — бинарная матрица. Каждое число является либо единицей, либо нулём.

**Сумма $n$ по всем наборам входных данных в одном тесте не превосходит $500$.**

## Выходные данные

Для каждого набора входных данных в отдельной строке выведите:

- **«YES»** — если заданная бинарная матрица является матрицей смежности некоторого неориентированного графа.  
  В следующей строке выведите $n$ чисел, разделённых через пробел. $i$-е число означает степень $i$-й вершины;

- **«NO»** — в противном случае.


## Пример
**входные данные**
```
2
2
0 1
1 0
3
0 0 1
1 0 0
0 0 0
```

**выходные данные**
```
YES
1 1 
NO 
```
