# D. Разрезание графа

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 256 мегабайт

Дан неориентированный граф. Над ним в заданном порядке производят операции следующих двух типов:

- $\text{cut } u\ v$ — удалить из графа ребро $u - v$;
- $\text{ask } u\ v$ — проверить, лежат ли две вершины $u$ и $v$ в одной компоненте связности.

Известно, что после выполнения всех операций типа $\text{cut}$, рёбер в графе не осталось. Найдите результат выполнения
каждой из операций типа $\text{ask}$.

## Входные данные

Первая строка входного файла содержит три целых числа, разделённые пробелами:
$$
n,\ m,\ k \quad (1 \le n \le 50\,000,\ 0 \le m \le 100\,000,\ m \le k \le 150\,000)
$$
— количество вершин графа, количество рёбер и количество операций, соответственно.

Следующие $m$ строк задают рёбра графа; $i$-я из этих строк содержит два числа
$$
u_i,\ v_i \quad (1 \le u_i, v_i \le n)
$$
— номера концов $i$-го ребра. Вершины нумеруются с единицы; граф не содержит петель и кратных рёбер.

Далее следуют $k$ строк, описывающих операции.

Операция типа $\text{cut}$ задаётся строкой:
$$
\text{cut } u\ v \quad (1 \le u, v \le n)
$$
которая означает, что из графа удаляют ребро между вершинами $u$ и $v$.

Операция типа $\text{ask}$ задаётся строкой:
$$
\text{ask } u\ v \quad (1 \le u, v \le n)
$$
которая означает, что необходимо узнать, лежат ли в данный момент вершины $u$ и $v$ в одной компоненте связности.

Гарантируется, что каждое ребро графа встретится в операциях типа $\text{cut}$ ровно один раз.

## Выходные данные

Для каждой операции $\text{ask}$ во входном файле выведите на отдельной строке слово:

- $\text{YES}$, если две указанные вершины лежат в одной компоненте связности;
- $\text{NO}$ в противном случае.

Порядок ответов должен соответствовать порядку операций $\text{ask}$ во входном файле.

### Примеры

**Ввод**

```
3 3 7
1 2
2 3
3 1
ask 3 3
cut 1 2
ask 1 2
cut 1 3
ask 2 1
cut 2 3
ask 3 1
```

**Вывод**

```
YES
YES
NO
NO
```
