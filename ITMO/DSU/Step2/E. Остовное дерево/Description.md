# E. Остовное дерево

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 256 мегабайт

Требуется найти в связном графе остовное дерево минимального веса.

**Входные данные**

Первая строка входного файла содержит два натуральных числа $n$ и $m$ ($2 \le n \le 200\,000$, $1 \le m \le 200\,000$) —
число вершин и ребер, соответственно. Следующие $m$ строк содержат описание ребер по одному на строке. Ребро номер $i$
описывается тремя натуральными числами $b_i$, $e_i$ и $w_i$ — номера концов ребра и его вес
соответственно ($1 \le b_i, e_i \le n$, $0 \le w_i \le 100\,000$).

Граф является связным.

**Выходные данные**

Первая строка выходного файла должна содержать одно натуральное число — вес минимального остовного дерева.

### Примеры

**Ввод**

```
4 4
1 2 1
2 3 2
3 4 5
4 1 4
```

**Вывод**

```
7
```
