# I. Двудольный граф

* ограничение по времени на тест 2 секунды
* ограничение по памяти на тест 256 мегабайт

Задан неориентированный граф на $n$ вершинах без рёбер. Каждая вершина графа в каждый момент времени покрашена в один из
цветов: $0$ или $1$ — и верно, что цвета вершин, между которыми есть ребро, различны.

Поступают два вида запросов:

1. Заданы два элемента $x$ и $y$ из разных компонент связности: добавить в граф ребро $(x, y)$, изменив цвета вершин для
   соответствия условию.
2. Заданы два элемента $x$ и $y$ из одной компоненты связности: ответить, правда ли, что их цвета совпадают.

Граф изначально пустой.

## Входные данные

В первой строке содержится два целых числа $n$ и $m$ ($1 \le n, m \le 2 \cdot 10^5$) — число вершин в графе и число
запросов, соответственно. Вершины нумеруются с нуля.

В последующих $m$ строках содержится описание запросов.

- Запрос первого вида имеет вид: «0 $a$ $b$» ($1 \le a, b \le n$), означающий, что нужно объединить компоненты с
  вершинами $x$ и $y$, такие что  
  $x \bmod n = (a + \text{shift}) \bmod n$ и $y \bmod n = (b + \text{shift}) \bmod n$.
- Запрос второго типа имеет вид: «1 $a$ $b$» ($1 \le a, b \le n$), означающий, что нужно проверить различность цветов
  вершин $x$ и $y$, таких что  
  $x \bmod n = (a + \text{shift}) \bmod n$ и $y \bmod n = (b + \text{shift}) \bmod n$.

Если ответ на этот запрос — да, то нужно выполнить присвоение $\text{shift} = (\text{shift} + 1) \bmod n$.

Изначально $\text{shift}$ равен нулю.

## Выходные данные

Для каждого запроса второго типа выведите в отдельной строке: «YES», если цвета вершин совпадают, иначе, «NO».

### Пример 1

#### Ввод

```
3 5
0 1 2
0 2 3
1 1 2
1 1 3
1 1 3
```

#### Вывод

```
NO
YES
NO
```
