# C. Задача динамической связности в офлайне

* ограничение по времени на тест 4 секунды
* ограничение по памяти на тест 256 мегабайт

Вам дан пустой граф на $n$ вершинах. Нужно ответить на запросы трёх типов:

- «+ $u$ $v$» — добавить неориентированное ребро $u$‑$v$ в граф.
- «- $u$ $v$» — удалить неориентированное ребро $u$‑$v$ из графа.
- «?» — посчитать число компонент связности в графе.

## Входные данные

В первой строке задано два числа $n$ и $m$ ($1 \le n \le 3 \cdot 10^5$, $0 \le m \le 3 \cdot 10^5$) — число вершин и
число запросов, соответственно. Каждая из следующих $m$ строк содержит один запрос. Для запроса на добавление
ребра $u$‑$v$ строка содержит «+ $u$ $v$» ($1 \le u \neq v \le n$). Гарантируется, что в момент запроса граф не содержит
такое ребро. Для запроса на удаление ребра $u$‑$v$ строка содержит «- $u$ $v$» ($1 \le u \neq v \le n$). Гарантируется,
что в момент запроса граф содержит такое ребро. Для запроса на подсчёт числа компонент связности строка содержит «?».

## Выходные данные

Для каждого запроса «?» выведите число компонент связности в графе в отдельной строке.

### Пример 1

#### Ввод

```
5 11
?
+ 1 2
+ 2 3
+ 3 4
+ 4 5
+ 5 1
?
- 2 3
?
- 4 5
?
```

#### Вывод

```
5
1
1
2
```
