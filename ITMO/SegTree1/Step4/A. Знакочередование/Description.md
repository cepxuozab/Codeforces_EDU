# A. Знакочередование

Реализуйте структуру данных из n элементов a<sub>1</sub>,a<sub>2</sub>…a<sub>n</sub>, поддерживающую следующие операции:

* присвоить элементу a<sub>i</sub> значение j;
* найти знакочередующуюся сумму на отрезке от l до r включительно (a<sub>l</sub>−a<sub>l+1</sub>+a<sub>l+2</sub>
  −…±a<sub>r</sub>).

## Входные данные

В первой строке входного файла содержится натуральное число n (1≤n≤10<sup>5</sup>) — длина массива.  
Во второй строке записаны начальные значения элементов (неотрицательные целые числа, не превосходящие 10<sup>4</sup>).

В третьей строке находится натуральное число m (1≤m≤10<sup>5</sup>) — количество операций.

В последующих m строках записаны операции:

* операция первого типа задается тремя числами 0 i j (1≤i≤n, 1≤j≤10<sup>4</sup>).
* операция второго типа задается тремя числами 1 l r (1≤l≤r≤n).

## Выходные данные

Для каждой операции второго типа выведите на отдельной строке соответствующую знакочередующуюся сумму.

## Примеры

**Ввод**

```
3
1 2 3
5
1 1 2
1 1 3
1 2 3
0 2 1
1 1 3
```

**Вывод**

```
-1
2
-1
3
```
