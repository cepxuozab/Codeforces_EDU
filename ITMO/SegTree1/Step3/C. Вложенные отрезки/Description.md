## C. Вложенные отрезки

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Дан массив из $2n$ чисел, каждое число от $1$ до $n$ в нём встречается ровно два раза. Будем говорить, что отрезок $y$
вложен в отрезок $x$, если оба вхождения числа $y$ находятся между вхождениями числа $x$. Найдите для каждого
отрезка $i$, сколько есть отрезков, которые в него вложены.

### Входные данные

Первая строка содержит число $n$ ($1 \leq n \leq 10^5$), вторая строка содержит $2n$ чисел. Гарантируется, что каждое
число от $1$ до $n$ встречается ровно два раза.

### Выходные данные

Выведите $n$ чисел, $i$-е число равно числу отрезков, вложенных в отрезок $i$.

## Примеры

### Пример 1

**Входные данные:**

```
5
5 1 2 2 3 1 3 4 5 4
```

**Выходные данные:**

```
1 0 0 0 3 
```
