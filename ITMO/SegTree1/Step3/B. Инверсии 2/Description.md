## B. Инверсии 2

**Ограничение по времени на тест:** 1 секунда  
**Ограничение по памяти на тест:** 1024 мегабайта

Это задача, обратная предыдущей. Была перестановка $p_i$ из $n$ элементов, для каждого $i$ записали число $a_i$ — число
таких $j$, что $j < i$ и $p_j > p_i$. Восстановите по заданным $a_i$ исходную перестановку.

### Входные данные

Первая строка содержит число $n$ ($1 \leq n \leq 10^5$), вторая строка содержит $n$ чисел $a_i$. Гарантируется,
что $a_i$ были получены из какой-то перестановки с помощью процедуры, описанной в условии.

### Выходные данные

Выведите $n$ чисел, исходную перестановку.

## Примеры

### Пример 1

**Входные данные:**

```
5
0 1 1 0 3
```

**Выходные данные:**

```
4 1 3 5 2 
```
