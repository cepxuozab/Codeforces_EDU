# A. Дерево отрезков на сумму

В этой задаче вам нужно написать обычное дерево отрезков на сумму.

## Входные данные

Первая строка содержит два числа `n` и `m` ($1 \leq n, m \leq 100\,000$) — размер массива и число операций.  
Следующая строка содержит `n` чисел $a_i$ — начальное состояние массива ($0 \leq a_i \leq 10^9$).

Далее следует описание операций. Описание каждой операции имеет следующий вид:

- **`1 i v`** — присвоить элементу с индексом `i` значение `v` ($0 \leq i < n$, $0 \leq v \leq 10^9$).
- **`2 l r`** — вычислить сумму элементов с индексами от `l` до `r - 1` ($0 \leq l < r \leq n$).

## Выходные данные

Для каждой операции второго типа выведите соответствующую сумму.

## Примеры

### Пример 1

**Входные данные:**

```
5 5
5 4 2 3 5
2 0 3
1 1 1
2 0 3
1 3 1
2 0 5
```

**Выходные данные:**

```
11
8
14
```
