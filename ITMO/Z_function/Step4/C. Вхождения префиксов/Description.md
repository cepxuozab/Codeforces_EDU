# C. Вхождения префиксов

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

Задана строка $s$. Для каждого $i$ от $1$ до $|s|$ найдите количество вхождений в строку её префикса длины $i$.

## Входные данные

В первой строке входных данных записано целое число $q$ ($1 \le q \le 10^5$) — количество наборов входных данных в
тесте.

Каждый набор входных данных состоит из строки $s$. Длина строки $s$ от $1$ до $10^6$ символов. Строка состоит
исключительно из строчных букв латинского алфавита.

Сумма длин строк $s$ по всем $q$ наборам входных данных в тесте не превосходит $10^6$.

## Выходные данные

Для каждого набора входных данных выведите $|s|$ целых чисел $c_1, c_2, \dots, c_{|s|}$, где $c_i$ — количество
вхождений в строку $s$ префикса $s$ длины $i$.

## Пример

**Входные данные**

```text
5
abacaba
eeeee
abcdef
ababababa
kekkekkek
```

**Выходные данные**

```text
4 2 2 1 1 1 1 
5 4 3 2 1 
1 1 1 1 1 1 
5 4 4 3 3 2 2 1 1 
6 3 3 2 2 2 1 1 1 
```
