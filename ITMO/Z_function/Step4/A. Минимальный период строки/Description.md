# A. Минимальный период строки

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

В этой задаче строка $p$ называется периодом строки $s$, если повторённая бесконечное количество раз строка $p$
содержит $s$ в качестве своего префикса. Иными словами, если $pp\dots p\dots$ начинается с $s$.

Ваша задача найти такую строку $p$, которая является кратчайшим периодом заданной строки $s$. Например,
для $s = \text{abcab}$ её минимальный период это $p = \text{abc}$.

## Входные данные

В первой строке входных данных записано целое число $t$ ($1 \le t \le 10^5$) — количество наборов входных данных в
тесте.

Каждый набор входных данных состоит из строки $s$. Длина строки $s$ от $1$ до $10^6$ символов. Строка $s$ состоит
исключительно из прописных и строчных букв латинского алфавита и цифр.

Сумма длин строк $s$ по всем $t$ наборам входных данных в тесте не превосходит $10^6$.

## Выходные данные

Выведите минимальный период для каждого из наборов входных данных.

## Пример

**Входные данные**

```text
7
abcabcabca
ABACABADABACABA
aabaaaba
aaaaaaaaaaaa
abcdefg
xoxxoxx
abcdea
```  

**Выходные данные**

```text
abc
ABACABAD
aaba
a
abcdefg
xox
abcde
```
