# E. Странная операция

**Ограничение по времени на тест:** 0.5 секунд  
**Ограничение по памяти на тест:** 512 мегабайт

Для заданной строки $s = s_1 s_2 \dots s_n$ допустимая операция состоит из следующих шагов:

1. выберем целое число $k$ от $0$ до $n$, включительно;
2. удалим первые $k$ символов, но допишем их в конец строки в обратном порядке: то есть $s$ примет
   вид $s_{k+1} \dots s_n\ s_k\ s_{k-1} \dots s_1$.

Заданы две строки $s$ и $t$. Проверьте существует ли такая операция, что делает $s$ равной $t$. Вы можете применить
операцию только один раз.

## Входные данные

Входные данные состоят из двух строк — из строки $s$ и строки $t$. Их длины от $1$ до $50000$, длины могут быть
различны. Обе строки состоят только из букв латинского алфавита и цифр.

## Выходные данные

Выведите «Yes», если требуемая операция существует. В противном случае выведите «No». В случае положительного ответа
выведите подходящее $k$, если таких несколько — минимальное.

## Примеры

**Входные данные**

```text
wpwdwpw
wdwpwpw
```

**Выходные данные**

```text
Yes
2
```

**Входные данные**

```text
codeforces
forcesedoc
```

**Выходные данные**

```text
Yes
4
```

**Входные данные**

```text
zzzz
zzzzz
```

**Выходные данные**

```text
No
```
