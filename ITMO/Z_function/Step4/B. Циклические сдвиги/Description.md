# B. Циклические сдвиги

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

Задана строка $s$ длины $n$. Её циклическим сдвигом влево на величину $k$ ($0 \le k < n$) называется строка $t$
вида $s_k s_{k+1} \dots s_{n-1} s_0 s_1 \dots s_{k-1}$.

Заданы две строки $s$ и $t$ одинаковой длины. Надо проверить, является ли $t$ циклическим сдвигом $s$. Если да, то
выведите соответствующее $k$ из определения выше.

## Входные данные

В первой строке входных данных записано целое число $q$ ($1 \le q \le 10^5$) — количество наборов входных данных в
тесте.

Каждый набор входных данных состоит из двух строк $s$ и $t$. Длина строки $s$ равна длине строки $t$ и находится в
границах от $1$ до $10^6$ символов. Строки $s$ и $t$ состоят исключительно из прописных и строчных букв латинского
алфавита и цифр.

Сумма длин строк $s$ по всем $q$ наборам входных данных в тесте не превосходит $10^6$.

## Выходные данные

Для каждого набора входных данных выведите `-1`, если строка $t$ не является циклическим сдвигом строки $s$. В противном
случае выведите величину любого циклического сдвига влево строки $s$, который равен $t$ — целое число от $0$ до $|s|$.

## Пример

**Входные данные**

```text
5
aaaabc
aaabca
abcd
acbd
bababa
ababab
r
r
itmo
oitm
```

**Выходные данные**

```text
1
-1
1
0
3
```
