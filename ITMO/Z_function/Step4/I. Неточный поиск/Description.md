# I. Неточный поиск

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

Задан текст $t = t_1 t_2 \dots t_{|t|}$ и образец $p = p_1 p_2 \dots p_{|p|}$. Дополнительно задан параметр $k$.

Если для индекса $d$ верно, что $t[d \dots d+|p|-1] = p$, то существует точное вхождение $p$ в $t$ с индекса $d$.

Для индекса $d$ в подстроке $t[d \dots d+|p|-1]$ существует такое окно (подстрока) из $f$ символов, что в этом окне
можно поменять $0$ или более символов так (но длина окна остаётся неизменной), что подстрока $t[d \dots d+|p|-1]$ станет
равна $p$ и при этом $f \le k$, то в этом случае говорят о неточном вхождении образца в текст. Очевидно, что точное
вхождение это частный случай неточного.

Например, если $k = 3$, $t = \text{«abacabadabacaba»}$ и $p = \text{«baxayad»}$, тогда есть одно неточное вхождение с
индекса $2$ (обратите внимание, что в этой задаче все индексы нумеруются от $1$).

Ваша задача найти все неточные вхождения образца $p$ в текст $t$.

## Входные данные

Первая строка содержит $t$. Вторая строка содержит $p$. Обе строки состоят из букв латинского алфавита. Их длины от $1$
до $10^6$, включительно. Третья строка содержит целое число $k$ ($0 \le k \le 10^6$).

## Выходные данные

Выведите количество неточных вхождений и индексы их начал в возрастающем порядке.

## Примеры

**Входные данные**

```text
abacaaa
aaa
1
```

**Выходные данные**

```text
4
1 3 4 5 
```

**Входные данные**

```text
ababa
aaaaa
3
```

**Выходные данные**

```text
1
1
```
