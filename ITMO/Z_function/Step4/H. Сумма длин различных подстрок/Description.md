# H. Сумма длин различных подстрок

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

Задана строка $s$. Выведите суммарную длину всевозможных различных подстрок этой строки.

## Входные данные

Единственная строка входных данных содержит $s$ — последовательность прописных и строчных латинских букв. Длина $s$ не
превышает $8000$ символов.

## Выходные данные

Выведите единственное целое число — суммарную длину всех различных подстрок.

## Примеры

**Входные данные**

```text
BOBR
```

**Выходные данные**

```text
19
```

**Входные данные**

```text
ABACABA
```

**Выходные данные**

```text
73
```  

**Примечание**  
Для первого примера все различные подстроки — это: B, O, R, BO, OB, BR, BOB, OBR, BOBR. Сумма их длин равна 19.
