# D. Палиндромный префикс

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Решите эту задачу с использованием **$z$-функции** строки. Существует много способов решить её, но в качестве упражнения
на использование $z$-функции попробуйте придумать и реализовать именно такое решение.

Задана строка $s$. Найдите длину её наидлиннейшего префикса, который является палиндромом.

## Входные данные

Входные данные состоят из строки $s$, которая содержит только строчные буквы латинского алфавита. Длина строки не
превосходит $4 \cdot 10^5$.

## Выходные данные

Выведите единственное число — длину наиболее длинного префикса $s$, который является палиндромом.

## Примеры

**Входные данные**

```text
aaab
```

**Выходные данные**

```text
1
```

**Входные данные**

```text
abab
```

**Выходные данные**

```text
3
```
