# A. Наидлиннейший палиндромный префикс - 1

*ограничение по времени на тест:* 2 секунда  
*ограничение по памяти на тест:* 512 мегабайт

_В дальнейшем эта задача будет предложена в больших ограничениях. Пока же для ознакомления с терминологией эту задачу вам
предлагается решить в небольших ограничениях._

Задана строка s. Найдите длину её наиболее длинного префикса, который является палиндромом.

Напомним, что палиндромом называется такая строка, которая одинаково читается как слева направо, так и справа налево.
Например, палиндромом является пустая строка, а также следующие строки: «wow», «abacaba», «z», «fffffff».

**Входные данные**  
В первой строке входных данных записано целое число t(1 &le; t &le; 100) — количество наборов входных данных в тесте.

Каждый набор входных данных состоит из единственной строки s, длина строки s от 1 до 2000 символов. Строка s состоит
исключительно из строчных букв латинского алфавита.

Сумма длин строк s по всем t наборам входных данных в тесте не превосходит 2000.

**Выходные данные**  
Выведите t целых чисел — для каждой строки выведите длину её наиболее длинного префикса, который является палиндромом.

## Примеры

### Пример 1

**Входные данные:**

```
8
woweffect
abccbaabc
testme
strstr
ababab
abcdefg
tetatet
aaaaaaaaaaaaa
```

**Выходные данные:**

```
3
6
1
1
5
1
7
13
```
