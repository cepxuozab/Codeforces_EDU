### C. Строка по z-функции

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Эта задача не требует знания эффективного алгоритма нахождения z-функции. Достаточно только понимания определения, чтобы
решить эту задачу.

Задан массив $z$ длины $n$ ($1 \le n \le 50$) из целых чисел от $0$ до $50$. Определите, существует ли такая строка $s$,
что её z-функция равна $z$. Если такая строка существует, то выведите любую такую строку. Используйте прописные и
строчные буквы латинского алфавита в качестве символов строки $s$.

#### Входные данные

В первой строке входных данных записано целое число $t$ ($1 \le t \le 10^4$) — количество наборов входных данных в
тесте.

Каждый набор входных данных состоит из двух строк. В первой строке записано целое число $n$ ($1 \le n \le 50$) — длина
массива $z$. Во второй строке содержатся целые числа $z_0, z_1, \dots, z_{n-1}$ ($0 \le z_i \le 50$).

#### Выходные данные

Выведите $t$ строк. Очередная строка должна содержать «!», если искомой строки $s$ не существует. В противном случае
выведите любую подходящую строку $s$, которая состоит из прописных и строчных букв латинского алфавита.

#### Пример

**Входные данные**

```text
5
15
0 0 1 0 3 0 1 0 7 0 1 0 3 0 1
6
0 0 0 0 0 0
5
0 4 3 2 1
1
50
2
1 1
```

**Выходные данные**

```text
abacabazabacaba
abcdef
eeeee
!
!
```
