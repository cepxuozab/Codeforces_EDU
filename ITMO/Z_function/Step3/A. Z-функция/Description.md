# A. Z-функция (простая версия)

*ограничение по времени на тест:* 3 секунды  
*ограничение по памяти на тест:* 512 мегабайт



Напишите программу, которая выведет z-функцию заданной строки `s`.

## Входные данные

Входные данные содержат единственную строку `s` длины от `1` до $10^8$. Она содержит только строчные и прописные буквы
латинского алфавита и цифры.

## Выходные данные

Выведите z-функцию строки — последовательность целых чисел $z_0, z_1, ..., z_n-1$, где $z_0 = 0$ и $z_i$ — это длина
наидлиннейшего общего префикса строки `s` и $s[i…n-1]$ для `i > 0` (считайте, что символы в строке пронумерованы от `0`
до `n-1`).

## Примеры

**Входные данные**

```
abacaba
```

**Выходные данные**

```
0 0 1 0 3 0 1
```

**Входные данные**

```
a
```

**Выходные данные**

```
0
```

**Входные данные**

```
aabbb
```

**Выходные данные**

```
0 1 0 0 0
```

**Входные данные**

```
babaabbb
```

**Выходные данные**

```
0 0 2 0 0 1 1 1
```
