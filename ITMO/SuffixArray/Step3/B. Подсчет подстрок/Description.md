# B. Подсчет подстрок

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Попробуйте модифицировать метод, чтобы решить следующую задачу.

Дана строка $t$ и $n$ запросов, каждый запрос — это некоторая строка $s_i$. Требуется для каждого запроса определить,
сколько раз строка $s_i$ встречается в $t$ как подстрока.

## Входные данные

Первая строка входных данных содержит строку $t$ ($1 \le |t| \le 300,000$).

Вторая строка содержит целое число $n$ — число запросов ($1 \le n \le 300,000$). Следующие $n$ строк содержат по одной
строке $s_i$. Сумма длин всех строк $s_i$ не превосходит $300,000$.

Все строки состоят из строчных латинских букв.

## Выходные данные

Для каждого запроса выведите одно число — сколько раз встречается строка $s_i$ в строке $t$.

## Примеры

**Пример 1**

Входные данные:

```text
ababba
3
ba
baba
abba
```

Выходные данные:

```text
2
0
1
```

**Пример 2**

Входные данные:

```text
itmouniversity
3
it
more
university
```

Выходные данные:

```text
2
0
1
```

**Пример 3**

Входные данные:

```text
aaa
2
a
aa
```

Выходные данные:

```text
3
2
```
