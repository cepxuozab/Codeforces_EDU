# A. Поиск подстрок

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Реализуйте алгоритм, который мы разобрали на лекции.

Дана строка $t$ и $n$ запросов, каждый запрос — это некоторая строка $s_i$. Требуется для каждого запроса определить,
встречается ли строка $s_i$ в $t$ как подстрока.

## Входные данные

Первая строка входных данных содержит строку $t$ ($1 \le |t| \le 300,000$).

Вторая строка содержит целое число $n$ — число запросов ($1 \le n \le 300,000$). Следующие $n$ строк содержат по одной
непустой строке $s_i$. Сумма длин всех строк $s_i$ не превосходит $300,000$.

Все строки состоят из строчных латинских букв.

## Выходные данные

Для каждого запроса выведите «Yes», если строка $s_i$ встречается в $t$ и «No» в противном случае.

## Примеры

**Пример 1**

Входные данные:

```text
ababba
3
ba
baba
abba
```

Выходные данные:

```text
Yes
No
Yes
```

**Пример 2**

Входные данные:

```text
codeforces
3
code
forces
math
```

Выходные данные:

```text
Yes
Yes
No
```

**Пример 3**

Входные данные:

```text
a
2
a
a
```

Выходные данные:

```text
Yes
Yes
```
