# A. Суффиксный массив - 1

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Теперь попробуйте самостоятельно реализовать алгоритм, который был разобран на лекции.

Напишите программу, которая считывает строку $s$ и выводит для нее суффиксный массив $p$.

## Входные данные

Первая строка входных данных содержит строку $s$ длины $n$ ($1 \le n \le 10^5$). Строка состоит из строчных букв
латинского алфавита.

## Выходные данные

Выведите $n+1$ число — номера первых символов суффиксов строки $s$ в отсортированном порядке.

## Примеры

**Входные данные**

```text
ababba
```

**Выходные данные**

```text
6 5 0 2 4 1 3 
```

**Входные данные**

```text
aaaa
```

**Выходные данные**

```text
4 3 2 1 0 
```  

**Входные данные**

```text
ppppplppp
```

**Выходные данные**

```text
9 5 8 4 7 3 6 2 1 0 
```  

**Входные данные**

```text
nn
```

**Выходные данные**

```text
2 1 0 
```
