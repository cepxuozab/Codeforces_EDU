# A. Суффиксный массив и LCP

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Попробуйте сами реализовать алгоритм Касаи, Аримуры, Арикавы, Ли и Пака, и добавить к построенному на прошлом шаге
суффиксному массиву значения $lcp$.

Постройте суффиксный массив для заданной строки $s$, для каждых двух соседних суффиксов найдите длину максимального
общего префикса.

## Входные данные

Первая строка входных данных содержит строку $s$ длины $n$ ($1 \le n \le 400,000$). Строка состоит из строчных латинских
букв.

## Выходные данные

В первой строке выведите $n+1$ число — номера первых символов суффиксов строки $s$ так, чтобы соответствующие суффиксы
были упорядочены в лексикографически возрастающем порядке.

Во второй строке выведите $n$ чисел — длины наибольших общих префиксов.

## Примеры

**Пример 1**

Входные данные:

```text
ababba
```

Выходные данные:

```text
6 5 0 2 4 1 3 
0 1 2 0 2 1 
```

**Пример 2**

Входные данные:

```text
aaaa
```

Выходные данные:

```text
4 3 2 1 0 
0 1 2 3 
```

**Пример 3**

Входные данные:

```text
ppppplppp
```

Выходные данные:

```text
9 5 8 4 7 3 6 2 1 0 
0 0 1 1 2 2 3 3 4 
```

**Пример 4**

Входные данные:

```text
nn
```

Выходные данные:

```text
2 1 0 
0 1 
```
