# C. Сортировка подстрок

**Ограничение по времени на тест:** 5 секунд  
**Ограничение по памяти на тест:** 256 мегабайт

Задана строка $s$ и набор её подстрок. Подстроки определяются парами чисел $l_i, r_i$ — позицией левого конца и правого
конца подстроки. Требуется отсортировать подстроки в лексикографическом порядке (как в словаре). Если подстроки равны
как слова, то в качестве дополнительного критерия порядка используйте сами пары (т.е. в первую очередь сравнивайте
по $l$, затем по $r$).

## Входные данные

В первой строке записана непустая строка $s$. Её длина не превосходит $4 \cdot 10^5$, она может содержать любые символы
с кодами от 33 до 127.

Вторая строка содержит $n$ ($1 \le n \le 4 \cdot 10^5$) — количество подстрок в наборе. Следующие $n$ строк содержат по
паре целых чисел $l_i, r_i$ ($1 \le l_i \le r_i \le |s|$), где $|s|$ — длина строки $s$. Возможно, что набор подстрок
содержит одинаковые пары.

## Выходные данные

Выведите $n$ строк, каждая строка содержит описание подстроки в формате как во входных данных. Порядок подстрок
определяется условием задачи.

## Примеры

**Пример 1**

Входные данные:

```text
abacaba
6
4 7
1 2
1 1
3 4
5 7
1 3
```

Выходные данные:

```text
1 1
1 2
1 3
5 7
3 4
4 7
```

**Пример 2**

Входные данные:

```text
a
1
1 1
```

Выходные данные:

```text
1 1
```

**Пример 3**

Входные данные:

```text
aa
5
1 1
1 2
2 2
1 1
2 2
```

Выходные данные:

```text
1 1
1 1
2 2
2 2
1 2
```
