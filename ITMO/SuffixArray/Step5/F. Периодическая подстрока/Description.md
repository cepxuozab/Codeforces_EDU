# F. Периодическая подстрока

Дана строка `s`. Найдите такое максимальное `k`, что в строке `s`есть непустая подстрока, которая является конкатенацией
`k`одинаковых строк.

### Входные данные

Первая строка входных данных содержит строку `s`($1 \le; |s| \le; 400000$). Строка состоит из строчных латинских букв.

### Выходные данные

Выведите искомое значение `k`.

## Примеры

**Ввод**

```
aabaaabaaabaaabaab
```

**Вывод**

```
3
```

**Ввод**

```
aabaabb
```

**Вывод**

```
2
```

**Ввод**

```
aaaa
```

**Вывод**

```
4
```

**Вывод**

```
2
```

**Ввод**

```
ppppplppp
```

**Вывод**

```
5
```

**Ввод**

```
nn
```

**Вывод**

```
2
```

### Примечание

В первом примере подстрока, которая повторяется `4`раза равна «abaa».
