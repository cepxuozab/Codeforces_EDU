# A. Число различных подстрок

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Посчитайте число различных подстрок заданной строки $s$.

## Входные данные

Единственная строка входных данных содержит строку $s$ длины $n$ ($1 \le n \le 400,000$). Строка состоит из строчных
латинских букв.

## Выходные данные

Выведите число различных подстрок в заданной строке.

## Примеры

**Пример 1**

Входные данные:

```text
ababba
```

Выходные данные:

```text
15
```

**Пример 2**

Входные данные:

```text
mmuc
```

Выходные данные:

```text
9
```

**Пример 3**

Входные данные:

```text
xmnnnuu
```

Выходные данные:

```text
24
```

**Пример 4**

Входные данные:

```text
nnnn
```

Выходные данные:

```text
4
```
