# D. Грани

**Ограничение по времени на тест:** 2 секунды  
**Ограничение по памяти на тест:** 512 мегабайт

Строка $t$ является гранью строки $s$, если она одновременно является ее префиксом и суффиксом и отлична от строки $s$.
Например, у строки «abbabba» три грани: «abba», «a» и пустая строка.

Задана строка $s$, посчитайте суммарное количество граней у всех ее подстрок. Каждую подстроку надо учитывать столько
раз, сколько раз она встречается в $s$.

## Входные данные

Первая строка входных данных содержит строку $s$ ($1 \le |s| \le 400,000$). Строка состоит из строчных латинских букв.

## Выходные данные

Выведите одно число — суммарное количество граней у всех подстрок строки $s$.

## Примеры

**Пример 1**

Входные данные:

```text
ababb
```

Выходные данные:

```text
20
```

**Пример 2**

Входные данные:

```text
hhya
```

Выходные данные:

```text
11
```

**Пример 3**

Входные данные:

```text
aaazaaa
```

Выходные данные:

```text
50
```

**Пример 4**

Входные данные:

```text
a
```

Выходные данные:

```text
1
```

## Примечание

В первом примере $s = $«ababb». Ответ состоит из следующих слагаемых:

1 (учли подстроку $s[1 \dots 1] = $«a»): пустая строка — единственная грань;  
1 (учли подстроку $s[2 \dots 2] = $«b»): пустая строка — единственная грань;  
1 (учли подстроку $s[3 \dots 3] = $«a»): пустая строка — единственная грань;  
1 (учли подстроку $s[4 \dots 4] = $«b»): пустая строка — единственная грань;  
1 (учли подстроку $s[5 \dots 5] = $«b»): пустая строка — единственная грань;  
1 (учли подстроку $s[1 \dots 2] = $«ab»): пустая строка — единственная грань;  
1 (учли подстроку $s[2 \dots 3] = $«ba»): пустая строка — единственная грань;  
1 (учли подстроку $s[3 \dots 4] = $«ab»): пустая строка — единственная грань;  
2 (учли подстроку $s[4 \dots 5] = $«bb»): грани — «b» и пустая строка;  
2 (учли подстроку $s[1 \dots 3] = $«aba»): грани — «a» и пустая строка;  
2 (учли подстроку $s[2 \dots 4] = $«bab»): грани — «b» и пустая строка;  
1 (учли подстроку $s[3 \dots 5] = $«abb»): пустая строка — единственная грань;  
2 (учли подстроку $s[1 \dots 4] = $«abab»): грани — «ab» и пустая строка;  
2 (учли подстроку $s[2 \dots 5] = $«babb»): грани — «b» и пустая строка;  
1 (учли подстроку $s[1 \dots 5] = $«ababb»).
